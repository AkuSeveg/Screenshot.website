Developer API

Integrasikan layanan ini ke aplikasi Anda.

POST /api/screenshot
Parameter Wajib Deskripsi
url YA URL target (wajib diawali http:// atau https://)
device_scale TIDAK Kualitas render. Opsi: 1 (Standard), 2 (HD), 3 (4K). Default: 2.
full_page TIDAK Boolean (true/false) untuk mengambil screenshot satu halaman penuh.
Request Payload (JSON)
{
  "url": "https://www.apple.com",
  "width": 1920,
  "height": 1080,
  "device_scale": 2,
  "full_page": false
}
Success Response (JSON)
{
  "success": true,
  "image_url": "https://cloud-link...",
  "extracted_text": "iPhone 15 Pro...",
  "creator": "Yoanz"
}
